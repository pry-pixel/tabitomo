# 🧳 たびとも

旅の計画を、じぶんでも、みんなでも。ポップな旅のしおりアプリ。

- 📖 日程タイムライン（旅のしおり）
- 📍 行きたい場所リスト＋地図（Googleマップのリンク／Takeout CSV取り込み対応）
- 🌏 海外旅行の時差対応（現地時間⇄日本時間の切り替え表示）
- 👥 リンク共有でリアルタイム共同編集（Firebase）
- 📱 iPhoneは「ホーム画面に追加」でアプリのように使えるPWA

## 構成

静的Webアプリ（ビルド不要）。`index.html` をそのまま配信するだけで動きます。

- Firebase未設定時：端末内保存のローカルモード
- `js/firebase-config.js` に設定を入れるとクラウドモード（同期・共有が有効化）
- Firestoreのセキュリティルールは `firestore.rules` を使用
